## Sobre o Projeto

**Condicionais 2** é uma coleção de exercícios desenvolvidos para praticar estruturas condicionais em Python, abrangendo desde situações introdutórias até cenários mais próximos de regras de negócio encontradas em aplicações reais.

Os exercícios exploram tomada de decisão, validações, classificações, controle de acesso, análise de dados e aplicação de múltiplas regras utilizando `if`, `elif` e `else`.

Além das condicionais, algumas atividades utilizam conceitos básicos da linguagem, como operadores lógicos, manipulação de strings, cálculos matemáticos e entrada de dados pelo usuário, sempre mantendo as estruturas condicionais como foco principal da resolução.

O projeto possui caráter educacional e foi desenvolvido como parte dos estudos de lógica de programação e fundamentos da linguagem Python.

## Objetivos de Aprendizagem

Este projeto foi criado para praticar e consolidar conhecimentos relacionados a:

- Estruturas condicionais (`if`, `elif`, `else`);
- Operadores relacionais;
- Operadores lógicos (`and`, `or`, `not`);
- Condições aninhadas;
- Validação de dados;
- Classificação de informações;
- Tomada de decisão baseada em múltiplas regras;
- Entrada e saída de dados;
- Aplicação de lógica em cenários reais;
- Resolução de problemas utilizando Python.
